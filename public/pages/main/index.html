Source Server
provides API interfaces for consuming some Open data.
Gifs

Search gifs from the famous Giphy provider.
/giphy/search?q=tag list top 25 gif for the searced tag
/giphy/random?q=tag shows a random gif for the tag.
/giphy provides a simple UI to play around with the provided services.

Movies

Search Movies, TV shows and Persons from the famous themoviedb.org .
Movies /movies/search?what=movie&q=shrek lists the shrek movies.
TV /movies/search?what=tv&q=friends lists shows that has 'friends' in their title.
Person /movies/search?what=person&q=Jennifer Aniston lists movies the person was part of.
/movies provides a simple UI to play around with the provided services.
